# NumPy Learning Journey & Student Marks Analyzer

## 📌 Overview

This repository documents my learning journey with **NumPy**, one of the most important Python libraries for numerical computing, data analysis, artificial intelligence, and machine learning.

As part of my learning process, I explored NumPy fundamentals and built a mini project called **Student Marks Analyzer** to apply the concepts in a practical way.

---

## 🎯 Learning Objectives

* Understand NumPy arrays and their advantages over Python lists.
* Learn array operations and numerical computations.
* Explore data filtering and analysis techniques.
* Build a mini project using real-world data.
* Create a strong foundation for Data Science and Machine Learning.

# 🚀 Mini Project: Student Marks Analyzer

## Project Description

A simple NumPy-based project that analyzes student marks and generates useful insights.

---

## Features

### ✅ Display Student Data

* Student names
* Student marks

### ✅ Performance Analysis

* Highest marks
* Lowest marks
* Average marks

### ✅ Student Insights

* Topper name
* Lowest scorer name
* Students scoring 90 or above
* Students scoring below average

### ✅ Class Statistics

* Total number of students

---

* Array Indexing
* Boolean Filtering
* Statistical Functions
* `argmax()`
* `argmin()`
* Conditional Selection
